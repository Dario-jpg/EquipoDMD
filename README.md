# Proyecto Colaborativo – Equipo DMD

## Descripción
Este proyecto ha sido desarrollado como práctica colaborativa de Git y GitHub.
El objetivo ha sido aprender a trabajar en equipo utilizando ramas, Pull Requests y resolución de conflictos.

## Flujo de trabajo
Durante el desarrollo del proyecto se ha trabajado con:
- Ramas independientes para cada funcionalidad
- Pull Requests para revisión de código
- Resolución manual de conflictos
- Fusión final en la rama `main`
